Contributors: Zakir Sajib
Tags: author, signature
Requires at least: 2.8
Tested up to: 4.0.1

This plugin appends the author's display name to blog posts or/and pages.

== Description ==

Appends the post author's display name to posts. There is an option to turn off the display on your own site.

== Installation ==

1. Upload the plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
